# Maroogr

Simple website for Maroogr, hosted on GitHub Pages.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., `maroogr`)
2. Push this folder to the repository:
   ```bash
   cd ~/Projects/maroogr
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/maroogr.git
   git push -u origin main
   ```
3. Go to repository Settings â†’ Pages
4. Set Source to "Deploy from a branch"
5. Select `main` branch and `/ (root)` folder
6. Save

Your site will be available at `https://YOUR_USERNAME.github.io/maroogr/`
